Hybrid Research Collaboration

Cairn.

A shared collaborator for the research group — where humans and AI contribute to one durable, inspectable memory of the work.

On the name

A cairn is a pile of stones travelers build along a trail. Each person who passes adds a stone. The marker persists for those who follow, points to the path, and remains long after any individual builder is gone.

It is collectively built, openly inspectable, and survives the absence of any single contributor — the same properties we want from a research group's accumulated knowledge.

What it is

A tool and design standard for hybrid human/AI research collaboration. Each project's instance — a cairn — is a git repository organized following Cairn's conventions, with optional layers (agents, indices, capture modules, dashboards) added on top.
